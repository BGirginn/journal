# CONTRIBUTING.md

## Branching
- main: release-ready
- develop: integration
- feature/*: feature branches

## PR Kuralları
- Küçük PR’lar
- Her PR:
  - test ekle veya gerekçeyi yaz
  - perf etkisi varsa ölçüm koy
  - migration varsa MIGRATIONS.md güncelle

## Kod Stili
- Dart: `dart format`
- Lint: `flutter analyze` temiz olmalı

## Commit Mesajı
- feat: ...
- fix: ...
- refactor: ...
- perf: ...
- docs: ...
